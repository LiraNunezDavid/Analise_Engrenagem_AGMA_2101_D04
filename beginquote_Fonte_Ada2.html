O dente coroado pode ajudar a concentrar a carga em direção do centro do dente, mitigando ruído e vibração. Mas a coroação excessiva pode aumentar a tensão por contato numa região pontual e ocasionar falhas que afetam a vida útil remanescente do sistema \citep{Nu_ez_2017,Nu_ez_2018}.
O fator de proporção do pinhão, \(K_{\text{Hpf}}\), responde à deflexões no dente devido à carga. Essa deflexão é normalmente mais elevada para grandes larguras na face do dente. O \(K_{\text{Hpf}}\) pode ser encontrado com a relação da largura da face (b) e o diâmetro primitivo (Dp), apresentada a seguir.\[\begin{equation} K_{\text{Hpf}}\left\{\begin{matrix} \text{Se}\ b\leq 25\ \text{mm},\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \text{ent}ao\ \frac{b}{10\cdot D_{P}}-0,025\\\\\text{Se}\ 25\ <b\leq 432\ \text{mm},\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \text{ent}ao\ \frac{b}{10\cdot D_{P}}-0,0375+0,000492\cdot b\\\\\text{Se}\ 432\ <b\leq 1\ 020,\ \text{ent}ao\ \frac{b}{10\cdot D_{P}}-0,1109+0,000815\cdot b-0,000000353\cdot b^{2}\\ \end{matrix}\right.\ \nonumber \\ \end{equation}\]
Para valores de \(\frac{b}{10\cdot D_{P}}\) inferiores a 0,05, assuma o \(K_{\text{Hpf}}\) igual a 0,05.
