Na Tabela 9 são apresentados os principais fatores considerados para cálculo de tensão à flexão e ao contato na norma AGMA e o seu equivalente no software Solid Edge. O fator de sobrecarga no Solid Edge é chamado de fator de aplicação e ambos são iguais. O fator de distribuição de carga no Solid Edge é chamado de fator de montagem, sendo diferentes mesmo considerando os mesmos critérios de entrada de dados. O fator dinâmico possui o mesmo nome, mas, os valores são diferentes. No programa Solid Edge solicita a alimentação de um fator de aspereza que não existe na norma AGMA, podemos presumir que seja parte do fator dinâmico, mas não há informação para poder afirmar isso. O fator de tamanho possui o mesmo nome e seus valores são muito próximos. O coeficiente de elasticidade é chamado no Solid Edge de fator de elasticidade e ambos são muito próximos. Finalmente, para que a comparação seja mais consistente, com já foram encontrados os valores para o número de tensão permitido, tanto para flexão e contato, esses valores são alimentados no Solid Edge.
